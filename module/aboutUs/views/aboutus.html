Non-Functional Requirements Solution

The non-function requirements [NFR] for the system include the following

Availability

The system will ensure 99.9% availability required with the exception of predetermined maintenance windows.

Capacity

The system will be capable of scaling up as per the capacity planning done time to time.

Data Integrity

The system will be capable of maintain data integrity across all interfacing systems.

Performance

The system will be capable of complying with Response Time, Throughput, Utilization according to business need.

Scalability

The system will be capable to ensure easy addition of new data source(s) or modify input process from existing data sources, create or update business processes, add new data outputs or modify existing data outputs.
